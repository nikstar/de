Каркас фильтра получения глубины из плоско-параллельного движения для VirtualDub.

Вам понадобится Visual Studio 2015 Update 3.

В начале файла motion_estimator.hpp в константах FILTER_NAME и FILTER_AUTHOR укажите свои имя и фамилию. Также в настройках проекта Visual Studio можно для удобства вставить свои имя и фамилию в название выходного файла.

Разрешено менять только код, относящийся к классу MotionEstimator (файлы motion_estimator.hpp, motion_estimator.cpp) и к классу DepthEstimator (файлы depth_estimator.hpp, depth_estimator.cpp). В каркасе оценка движения реализована полным перебором в методе MotionEstimator::Estimate(), а оценка глубины - умножением длины вектора движения каждого пикселя на заданный множитель.

В папке Test находится одно из тестовых видео, а также скрипт для запуска и замера скорости работы фильтра. Каждый из замеров производится по пяти настройкам качества (20, 40, 60, 80, 100). Результаты замера скорости работы можно найти в файле DE_performance.log. Соответствующие получившиеся видео - в файлах depth-N.avi, где N - использованная настройка качества.

Для запуска скриптов нужно:
- Установить 32-битную версию VirtualDub.
- Скопировать ваш фильтр DE_your_surname.vdf в папку plugins32, находящуюся в папке, где установлен VirtualDub.
- В файле test.bat поменять путь к VirtualDub.exe.
- В файле test.vdscript подставить ваши имя и фамилию на 23-й строчке в DE_your_surname также, как вы это сделали в motion_estimator.hpp.

Скрипты настроены на файл source.avi. Если вы хотите настроить их на другой файл, поменяйте имя файла в первой строчке test.vdscript.
